1) Ativação do autoincremento nas tabelas.

A migração não trouxe as chaves primárias, então é ncessário criá-las. Para isso:

- Obtenha o próximo id para a tabela:
select max(id)+1 from suporteusuarioelemento;

- Crie o SEQUENCE com o valor retornado pela query:
CREATE SEQUENCE suporteusuarioelemento_seq START WITH 372;

- Ative o preenchimento do valor default pelo SEQUENCE:
ALTER TABLE suporteusuarioelemento ALTER COLUMN id SET DEFAULT nextval('suporteusuarioelemento_seq');
